This website is a small collection of browser games made in JavaScript.

Please note that this website uses outdated web standards in order to support the Nintendo 3DS browser. Also, not many things are possible compared to the modern browsers. Due to the browser using most buttons to control itself, it's not possible to use them all. They are detectable, but it's not possible to prevent the browser from triggering their default actions, example going to the previous page on .

If you'd like to suggest a game or create your own with JavaScript, feel free to suggest or open pull requests on the GitHub page.

This project is not affiliated with Nintendo, and it's only community driven.
